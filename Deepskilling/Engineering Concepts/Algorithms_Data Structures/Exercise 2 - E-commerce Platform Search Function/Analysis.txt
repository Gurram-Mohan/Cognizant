Big O Notation:
Big O notation is used to measure the efficiency of an algorithm as the input size increases.

Linear Search:
Best Case: O(1)
Average Case: O(n)
Worst Case: O(n)

Binary Search:
Best Case: O(1)
Average Case: O(log n)
Worst Case: O(log n)

Comparison:
Linear Search checks each element one by one.
Binary Search repeatedly divides the search space into halves.

Conclusion:
Binary Search is more suitable for an e-commerce platform when product data 
is sorted because it provides much faster search performance for large 
datasets.
